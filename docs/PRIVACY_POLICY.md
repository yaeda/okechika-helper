# プライバシーポリシー

最終更新日: 2026-03-14

本プライバシーポリシーは、ブラウザ拡張機能「Okechika Helper」（以下「本拡張機能」）における、ユーザー情報の取り扱いについて説明するものです。

## 1. 取得する情報

本拡張機能は、以下の情報をユーザー端末上で扱います。

- 解読テーブル（ユーザーが登録した文字対応）
- 解読テーブルの更新日時
- 対象ルートURL設定
- 発見済みページ情報（URL・タイトル・対象ルートURL）
- ブックマーク情報（URL・タイトル・対象ルートURL）

これらは、拡張機能の設定および機能提供のために使用されます。

## 2. 情報の保存場所

解読テーブル、対象ルートURL設定、各種設定、ブックマークは Chrome 拡張機能の `chrome.storage.sync` に保存されます。

発見済みページは Chrome 拡張機能の `chrome.storage.local` に保存されます。

## 3. 外部送信・第三者提供

本拡張機能は、ユーザー設定情報や解読テーブルを、開発者サーバー等の外部へ送信しません。

また、ユーザー情報を第三者へ販売・共有・提供することはありません。

## 4. 権限の利用目的

本拡張機能は、以下の権限を機能提供のために利用します。

- `storage`: 解読テーブル、設定情報、発見済みページ、ブックマークの保存・読み出し
- `scripting`: 許可された対象ページでのコンテンツスクリプト実行
- `sidePanel`: 対象ページ上でサイドパネルを開閉し、発見済みページ、相互変換、変換テーブルを表示するため
- `host permissions` / `optional host permissions`: ユーザーが利用対象として設定したサイト上で機能を有効化するため
- `favicon`（任意権限）: ユーザーが明示的に有効化した場合に、対象ルート URL 一覧で各サイトの favicon を表示するため

初期対象URL以外を追加する場合は、ユーザーの操作に応じてアクセス権限を要求します。

## 5. 個人情報の収集について

本拡張機能は、氏名、メールアドレス、位置情報、閲覧履歴などの個人情報を、開発者が受領する形で収集しません。

発見済みページは `chrome.storage.local`、ブックマークは `chrome.storage.sync` に保存されますが、いずれも開発者サーバー等へ送信されません。

## 6. 本ポリシーの変更

法令改正や機能変更に応じて、本ポリシーを改定する場合があります。改定後の内容は、本ドキュメントに反映します。
