この拡張は、対象ページ内、ブラウザの拡張機能ボタンから開くポップアップ、設定画面の 3 か所で使えます。

- 対象ページ内では、桶地下文字に対してルビが自動で表示され、桶地下文字を選択すると表示されるツールチップから、文字の対応の登録や検索ができます。
- ポップアップでは、発見済みページを確認し、その中でブックマークを管理できます。
- 設定画面では、それぞれのパネルで各種機能を使えます。

## 1. 対象ページ内で使う

桶地下文字には、対象サイト上で自動的にルビが表示されます。

対象サイト上で桶地下文字を選択すると、ツールチップが表示されます。

ツールチップでは次の操作ができます。

- 選択した桶地下文字に対応する内容を登録する
- 選択した桶地下文字をコピーする
- 変換後の文字列をコピーする
- 選択した桶地下文字でサイト内検索を行う

複数の桶地下文字をまとめて選択した場合は、選択した文字数に合わせて対応を登録します。未登録に戻したいときは `?` を入力します。

ボタンやリンク上の文字はそのままだと選択しにくいことがあります。文字を選択したいときは、`Alt` を押しながらドラッグしてください。

## 2. ポップアップで使う

ブラウザの拡張機能ボタンからポップアップを開けます。

ポップアップでは、現在開いている対象サイトに属する発見済みページを確認できます。

- `すべて` と `ブックマークのみ` を切り替えられます
- タイトルや URL で発見済みページを検索できます
- 一覧の項目を押すと、そのページを開けます
- 一覧からブックマークの追加・解除ができます
- 上部の `設定を開く` から設定画面を開けます

発見済みページは、対象サイトを閲覧している間に自動で記録されます。

現在開いているページが対象サイト外の場合は、発見済みページは表示されません。

## 3. 設定画面で使う

設定画面では、各パネルから機能を使えます。

上部には次のリンクがあります。

- `使い方`: 使い方の説明ページを開けます。
- `更新内容`: 更新内容のページを開けます。
- `不具合報告`: 不具合報告ページを開けます。

新しいバージョンがダウンロードされて適用待ちになったときは、設定画面上部の中央に更新通知が表示されます。`更新する` を押すと拡張を再読み込みして更新の反映を試みます。

### 対象ルート URL

- 拡張の対象にするサイトやパスを追加
- 追加した対象ルート URL の確認
- 不要になった対象ルート URL の削除

### 各種設定

- 24時間モードの ON / OFF
  - `qtes9gu0k.xyz` 向けの 24時間モードを切り替えます。
  - 24時間モードではツールチップや相互変換パネルの検索ボタンを時間制限なく使えます。
  - サイト右上の検索ボタンは時間制限されたままですのでご注意ください。
- ツールチップの検索ボタンを別タブで開くかどうかの切り替え
  - OFF の場合は現在のタブで検索結果を開きます。
  - ON の場合は新しいタブで検索結果を開きます。

### 相互変換

- 日本語から桶地下文字への変換
- 桶地下文字から日本語への変換
- 日本語から桶地下文字へ変換した結果をコピー
- 変換結果を使って公式サイトで検索

### 発見済みページ

- 対象サイトで見つけたページの一覧を確認
- タイトルや URL で発見済みページを検索
- URL ごとにページを開く
- ブックマークの追加・解除
- `すべて` と `ブックマークのみ` の切り替え

### 変換テーブル

- 登録済みの対応一覧を確認
- CSV のエクスポート
- CSV のインポート
- ダブルクリックで登録内容を直接編集
- 表示形式の切り替え

## 4. データについて

この拡張で保存されるデータには、同じ Google アカウントでログインしているブラウザ間で共有されるものと、そのブラウザだけに保存されるものがあります。

### 同じ Google アカウントでログインしているブラウザ間で共有されるもの

ブラウザの同期ストレージに保存されます。

- 変換テーブル
- 対象ルート URL
- 各種設定
- ブックマーク

### そのブラウザだけに保存されるもの

ブラウザごとのローカル保存になります。

- 発見済みページ
- 変換テーブルの表示形式
- 対象ルート URL の表示 / 非表示
- 相互変換パネルの選択タブ
- ポップアップの `すべて` / `ブックマークのみ` の選択
